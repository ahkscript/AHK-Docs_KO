SoundSetWaveVolume

사운드 장치에 대하여 웨이브 출력 볼륨을 변경합니다.

SoundSetWaveVolume, Percent [, DeviceNumber]

매개변수

Percent

퍼센트 숫자는 범위가 -100부터 100까지입니다 (부동 소수점이나 표현식을 허용합니다). 숫자가 플러스 또는 마이너스 사인으로 시작하면, 현재 볼륨 레벨이 지시한 양 만큼 위 또는 아래로 조절됩니다. 그렇지 않으면, 볼륨은 Percent로 지시한 레벨에 명시적으로 지정됩니다.

DeviceNumber

이 매개변수를 생략하면, 기본 값은 1입니다 (첫 번째 사운드 장치), 이 장치는 보통 녹음과 연주에 사용되는 시스템의 기본 장치입니다. 1보다 큰 값을 지정하면 다른 사운드 장치를 처리할 수 있습니다.

ErrorLevel

[v1.1.04+] 이 명령어는 실패시 예외를 던질 수 있습니다. 더 자세한 정보는 실행시간 에러를 참조하십시오.

ErrorLevel은 문제가 있었으면 1이 설정되고 그렇지 않으면 0이 설정됩니다.

논평

현재 웨이브 출력 볼륨 레벨은 SoundGetWaveVolume를 통하여 열람할 수 있습니다. Master Volume, Synth, Microphone, Mute, Treble, 그리고 Bass와 같은 설정은 SoundSet과 SoundGet를 통하여 각각 설정하고 열람할 수 있습니다.

Windows Vista 이후 (v1.1.10+): 이 명령어는 SoundSet에서 ComponentType은 Wave로 설정하고 ControlType는 Volume으로 설정한 것과 동등합니다. 두 명령어 모두 채널 사이의 기존의 평형을 유지하려고 노력합니다.

Windows 2000/XP/2003: SoundSet과 다르게, 이 명령어는 볼륨 레벨을 변경할 때 채널 사이의 기존의 평형을 유지하려고 노력합니다 (예, 왼쪽과 오른쪽).

관련 항목

SoundGetWaveVolume, SoundSet, SoundGet, SoundPlay

예제

SoundSetWaveVolume, 50  ; 볼륨을 그의 절반으로 설정합니다.
SoundSetWaveVolume, -10  ; 현재 레벨을 10만큼 낮춥니다 (예, 80이면 70이 됩니다).
SoundSetWaveVolume, +20  ; 현재 레벨을 20만큼 높입니다.
