ComObjError() [AHK_L 53+]

COM 에러의 고지를 켜고 끕니다.

Enabled := ComObjError([Enable])

Parameters

Enable

불리언 값 (true 또는 false). 선택적.

Enabled

[v1.0.91+]: 이 함수가 호출되기 전에 효과가 있던 값을받습니다.

총평

COM 객체에 접근하고 나면, A_LastError에는 COM 객체의 IDispatch::Invoke 함수가 돌려주는 HRESULT 코드가 담깁니다. 스크립트는 자신만의 에러-처리를 구현할 수 있습니다. ComObjError(false)를 호출하여 에러 고지를 끄고 A_LastError의 값을 참고하면 됩니다.

관련 항목

ComObjCreate, ComObjGet, ComObjActive, ComObjConnect
