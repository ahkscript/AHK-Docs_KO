DriveSpaceFree

드라이브의 여유 공간을 메가바이트 단위로 열람합니다.

DriveSpaceFree, OutputVar, Path

매개변수

OutputVar

결과를 저장할 변수, 온전한 숫자에 가장 가깝게 반내림됩니다.

path

정보를 수신할 드라이브 경로. NTFS는 볼륨 마운트와 디렉토리 연결을 지원하므로, 같은 "드라이브" 일 지라도 어떤 경우는 폴더마다 여유 공간을 다르게 보고할 수 있습니다.

논평

OutputVar는 메가바이트 단위로 디스크 여유 공간의 양이 설정됩니다 (가까운 메가바이트로 반내림됩니다).

관련 항목

Drive, DriveGet

예제

DriveSpaceFree, FreeSpace, c:\
