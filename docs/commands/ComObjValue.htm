ComObjValue() [v1.0.91+]

COM 포장 객체에 저장된 값이나 포인터를 열람합니다.

Value := ComObjValue(ComObject)

매개변수

Value

64-비트 부호있는 정수.

ComObject

COM 객체 또는 유형이 있는 값을 담고 있는 포장 객체.

총평

이 함수는 범용 목적이 아닙니다.

ComObjValue를 호출하는 것은 variant.llVal와 동등합니다. 여기에서 ComObject는 VARIANT 구조체로 취급됩니다. 이 함수를 사용하는 스크립트는 모두 반드시 어떤 유형의 값을 포장 객체가 가지고 있는지 그리고 어떻게 취급해야 할지 알고 있어야 합니다. 예를 들어, 인터페이스 포인터가 반환되면, Release는 호출하지 말아야 합니다. 그러나 AddRef는 스크립트가 그 포인터를 가지고 무엇을 하는가에 따라 필요할 수도 있습니다.

관련 항목

ComObjType, ComObjCreate, ComObjGet, ComObjActive
