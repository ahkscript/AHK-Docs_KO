FileGetTime

파일이나 폴더의 날짜시간 도장을 열람합니다.

FileGetTime, OutputVar [, Filename, WhichTime]

매개변수

OutputVar

열람된 날짜-시간을 저장할 변수의 이름. 형식은 YYYYMMDDHH24MISS입니다. 시간은 지역 시간이지, UTC/GMT가 아닙니다.

Filename

목표 파일이나 폴더의 이름. 절대 경로를 지정하지 않으면 %A_WorkingDir%에 있다고 간주합니다. 생략하면, 파일-회돌이의 가장 안쪽에 싸인 현재 파일이 대신 사용됩니다.

WhichTime
열람할 시간도장:
M = 변경시간 (매개변수를 생략하면 이것이 기본값)
C = 생성 시간
A = 마지막 접근 시간

ErrorLevel

[v1.1.04+] 이 명령어는 실패시 예외를 던질 수 있습니다. 더 자세한 정보는 실행시간 에러를 참조하십시오.

ErrorLevel은 문제가 있으면 1 없으면 0이 설정됩니다.

A_LastError는 운영 체제의 GetLastError() 함수의 결과가 설정됩니다.

논평

날짜와 시간에 관한 설정은 YYYYMMDDHH24MISS를 참조하십시오.

관련 항목

FileSetTime, FormatTime, If var is [not] type, FileGetAttrib, FileSetAttrib, FileGetSize, FileGetVersion, File-회돌이, EnvAdd, EnvSub

예제

FileGetTime, OutputVar, C:\My Documents\test.doc  ; 기본으로 변경 시간을 열람합니다.
FileGetTime, OutputVar, C:\My Documents\test.doc, C  ; 생성 시간을 열람합니다.
