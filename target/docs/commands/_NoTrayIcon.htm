#NoTrayIcon

스크립트 아무데서나 이 지시어를 지정하면 스크립트가 기동될 때 트레이 아이콘이 나타나는 것을 막습니다. (스크립트를 EXE로 컴파일해도 마찬가지입니다).

핫키를 가진 스크립트에 이를 사용하면, 핫키를 ExitApp 명령어에 묶는 게 좋습니다. 그렇지 않으면, 쉽게 프로그램을 빠져 나갈 방법이 없습니다 (컴퓨터를 재시작하거나 프로세스를 죽여야 합니다). 예를 들어: #x::ExitApp.

트레이 아이콘을 스크립트가 실행되는 동안 언제든지 감추거나 보여줄 수 있습니다. 다음 명령어 Menu, Tray, Icon 또는 Menu, Tray, NoIcon을 사용하면 됩니다. Menu, Tray, NoIcon을 스크립트의 맨 위에서 사용하면 단점이 있습니다. 스크립트가 처음으로 기동될 때 트레이 아이콘이 잠깐 보일 수 있습니다. 그를 피하려면, 대신에 #NoTrayIcon을 사용하십시오.

트레이 아이콘이 현재 보이지 않으면 내장 변수 A_IconHidden에 1이 담깁니다. 그렇지 않으면 0이 담깁니다.

관련 항목

Menu, ExitApp
