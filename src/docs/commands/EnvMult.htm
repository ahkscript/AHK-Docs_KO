EnvMult

주어진 값을 변수에 곱해 그 변수에 다시 설정합니다. 다음과 동등: Var *= Value.

EnvMult, Var, Value

매개변수

Var

연산을 수행할 변수의 이름.

Value

정수, 부동 수수점 수, 또는 표현식.

논평

이 명령어는 다음 단축 스타일과 동등합니다: Var *= Value.

Var나 Value가 비어 있거나 또는 숫자로 시작하지 않으면, 계산을 위해 0으로 간주됩니다 (단 내부적으로 표현식에 Var := X *= Y와 같이 사용할 경우는 제외합니다).

Var나 Value에 소수점이 포함되어 있으며, 그 결과는 부동 소수점 수가 됩니다. 형식은 SetFormat으로 설정됩니다.

관련 항목

EnvAdd, EnvSub, EnvDiv, SetFormat, 표현식, If var is [not] type, SetEnv, 비트별 연산 (변형)

예제

EnvMult, MyCount, 2 
MyCount *= 2  ; 위와 동등합니다
