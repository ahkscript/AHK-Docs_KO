ExitApp

스크립트를 무조건 종료합니다.

ExitApp [, ExitCode]

매개변수

ExitCode

-2147483648부터 2147483647 사이의 정수 ( v1.0.48.01+에서 표현식 가능). 이 값은 스크립트가 종료할 호출자에게 반환됩니다. 이 코드는 스크립트를 부화시킨 어느 프로그램도 접급할 수 있습니다. 예를 들어 (RunWait를 통한) 스크립트 또는 배치 파일(.bat). 생략하면, ExitCode는 기본값으로 0이 설정됩니다. 전통적으로 0은 성공을 나타내는 데 사용됩니다.

논평

스크립트는 즉시 종료합니다. OnExit 서브루틴이 있는 경우는 제외입니다. 이는 스크립트의 트레이 아이콘이나 메인 메뉴에서 "Exit"를 선택한 것과 동등합니다.

Exit 그리고 ExitApp는 스크립트에 핫키가 없고, 스크립트가 영구적이 아니며, 그리고 Num/Scroll/Capslock 키가 항상 AlwaysOn 또는 AlwaysOff를 유지하기를 요청하지 않는 경우는 행위가 동일합니다.

스크립트에 OnExit 서브루틴이 있다면, ExitApp에 응답하여 실행됩니다.

관련 항목

Exit, OnExit, #Persistent

예제

#x::ExitApp  ; 이 스크립트를 종료할 핫키를 할당합니다.
