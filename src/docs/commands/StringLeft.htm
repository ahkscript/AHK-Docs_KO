StringLeft / StringRight

문자열의 왼쪽이나 오른쪽으로부터 문자들을 열람합니다.

StringLeft, OutputVar, InputVar, Count
StringRight, OutputVar, InputVar, Count
NewStr := SubStr(String, StartPos [, Length]) ; 자세한 것은 SubStr() 함수를 참조하십시오.

Parameters

OutputVar

InputVar으로부터 추출한 부문자열을 저장할 변수의 이름.

InputVar

내용을 추출할 변수의 이름. 이름을 퍼센트 사인으로 둘러 싸지 마십시오. 단, 변수의 내용이 이름이기를 바라는 경우라면 예외입니다.

Count

추출할 문자의 개수, 표현식 가능. Count가 0보다 작거나 같으면, OutputVar는 비워집니다. Count가 InputVar의 길이를 초과하면, OutputVar는 InputVar 전체와 같게 설정됩니다.

논평

이 명령어와 다른 모든 명령어에 대하여, OutputVar는 InputVar와 같은 변수가 되어도 허용합니다.

관련 항목

SubStr(), StringMid, StringTrimLeft, StringTrimRight, IfInString, StringGetPos, StringLen, StringLower, StringUpper, StringReplace

예제

String = This is a test. 
StringLeft, OutputVar, String, 4  ; 문자열 "This"를 OutputVar에 저장합니다.
StringRight, OutputVar, String, 5  ; 문자열 "test."를 OutputVar에 저장합니다.
