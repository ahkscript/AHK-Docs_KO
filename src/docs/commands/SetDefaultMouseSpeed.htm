SetDefaultMouseSpeed

지정하지 않으면 Click 그리고 MouseMove/Click/Drag에 사용될 마우스 속도를 설정합니다.

SetDefaultMouseSpeed, Speed

매개변수

Speed

마우스를 이동하는 속도. 범위는 0 (가장 빠름)부터 100 (가장 느림)까지. 주의: 속도가 0이면 마우스를 순간 이동 시킵니다. 이 매개변수는 표현식일 수 있습니다.

논평

SendInput/Play 모드에 대하여 SetDefaultMouseSpeed는 무시됩니다; 이 모드에서는 마우스가 순간 이동합니다 (그렇지만, SetMouseDelay는 SendPlay에 적용되는 모드가 있습니다). 시각적으로 마우스를 더 느리게 이동하고 싶으면 -- 예를 들어 청중을 위하여 데모를 수행하는 스크립트라면 -- (선택적으로 BlockInput과 함께) SendEvent {Click 100, 200} 또는 SendMode Event를 사용하십시오.

이 명령억 사용되지 않으면, 기본 마우스 속도는 2입니다. 내장 변수 A_DefaultMouseSpeed에 현재 설정이 담겨 있습니다.

명령어 MouseClick, MouseMove, 그리고 MouseClickDrag는 모두 기본 마우스 속도를 오버라이드 할 매개변수가 있습니다.

Speed가 0 보다 크면, SetMouseDelay도 마우스가 목표를 향해 조금씩 이동할 때마다 지연 시간을 두어서 속도에 영향을 미칩니다.

새로 기동되는 쓰레드는 (예를 들어 핫키, 맞춤 메뉴 항목, 또는 타이머 서브루틴) 이 명령어에 대한 기본 설정을 가지고 새롭게 시작합니다. 이 기본값은 자동 실행 섹션 (스크립트의 상단)에서 이 명령어를 사용하여 변경할 수 있습니다.

관련 항목

SetMouseDelay, SendMode, Click, MouseClick, MouseMove, MouseClickDrag, SetWinDelay, SetControlDelay, SetKeyDelay, SetKeyDelay

예제

SetDefaultMouseSpeed, 0 ; 마우스를 순간 이동 시킵니다.
